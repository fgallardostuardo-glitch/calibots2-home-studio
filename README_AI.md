# Instrucciones para IA - Calibot

## Proyecto

Aplicacion interactiva para el estudio de ingles de ninos de kinder.

## Regla principal

No modificar las imagenes de la unidad 1.

## Direccion visual activa

- estilo: semirealista
- nivel de detalle: medio-alto
- fondo: sin fondo
- grosor de linea: delgado
- edad objetivo: 6 anos

## Alcance actual

- trabajar sobre la carpeta local del proyecto
- mantener consistencia visual entre unidades
- generar nuevos assets solo para las unidades 2 a 5, salvo indicacion explicita en contra

## Nota operativa

Antes de producir una tanda completa, crear y revisar un lote piloto para validar consistencia de estilo.
